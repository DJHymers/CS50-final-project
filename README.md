#### Description:
This is a simple 2D football/soccer prototype built in LÖVE2D for the CS50 final project. 

The game features:

- Player-controlled character with WASD movement.
- Opponent AI with possession logic, evasive dribbling, and goal attempts.
- Ball mechanics including dribbling, charging, shooting, passing, friction, and blocking.
- Pitch with goals, penalty boxes, center circle, and semi-transparent nets.
- Score tracking with goal messages, winner display, and max score game-over logic.
- Menu screen with difficulty selection (Easy, Medium, Hard).

Controls:

- **WASD** → Move player
- **Space** → Shoot ball
- **Left Shift** → Pass ball
- Menu keys **1, 2, 3** → Select difficulty

This prototype demonstrates basic physics, AI, and gameplay mechanics for a small football game.
